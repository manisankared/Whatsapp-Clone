WhatsApp Clone

Proyek ini adalah replika visual dari antarmuka aplikasi WhatsApp, yang dibangun menggunakan HTML, CSS, dan JavaScript. Proyek ini bertujuan untuk meniru desain dan tampilan WhatsApp secara mendetail serta memberikan saya wawasan dalam pengembangan antarmuka web.

Key Features

  • Mengganti judul halaman sesuai keinginan pengguna.
  • Dapat mengubah nama kontak
  • Mengedit pesan yang sudah dikirimkan.
  • Menyesuaikan waktu pengiriman pesan.
  • Mengubah status tanda centang (single, double, atau centang biru).
  • Memungkinkan tampilan dalam mode layar penuh (Full Screen).
  • Menyediakan opsi Mode Gelap dan Mode Terang untuk kenyamanan visual.
  • Dapat mengunggah status
